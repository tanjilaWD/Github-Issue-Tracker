1. var, let, const → var function scope can be redeclared, let block scope can be reassigned, const block scope but value cannot be changed.

2. Spread operator (...) → Used to expand or copy the elements of an array or object.

3. map(), filter(), forEach() → map() creates a new array, filter() takes elements according to the condition, forEach() just runs the loop.

4. Arrow function → JavaScript function of short syntax written using =>.

5. Template literals → Way to insert variable ${} inside a string using Backtick ( ).
